# Tourare Travel Landing Page

A premium editorial travel experiences landing page.

## Features
- **Modern Minimalist Design**: Clean layout focusing on high-quality imagery and typography.
- **Responsive**: Fully responsive across mobile, tablet, and desktop.
- **Dark Mode**: Supports system theme and manual toggle with persistence.
- **Glassmorphism**: Subtle glass effects on the navigation bar and buttons.
- **Micro-animations**: Smooth entry animations for a premium feel.

## How to use
1. Open `index.html` in your favorite browser.
2. To modify styles, you can adjust the Tailwind configuration in the `<script>` tag.
